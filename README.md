🚀 Trinetra PPE Detection

Trinetra PPE Detection is an AI-powered Computer Vision system designed to monitor industrial safety in real time. The system uses a YOLO-based object detection model to identify safety violations such as missing helmets and safety vests through live camera feeds.

This project is a part of the Trinetra AI vision — building an intelligent industrial safety monitoring platform.

🎯 Features
Real-time webcam-based detection
Helmet compliance detection
Safety vest compliance detection
PPE violation alert system
Clean bounding box visualization
Mask detection ignored (focused PPE logic)
GPU accelerated inference (CUDA supported)

🧠 How It Works
Captures live video from camera
Processes frames using YOLO model
Detects PPE-related objects
Applies rule-based logic:
If helmet or vest missing → Violation
Else → Safe
Displays result in real-time
🛠️ Tech Stack
Python
OpenCV
YOLOv8 (Ultralytics)
PyTorch (CUDA GPU support)

📌 Future Scope
Person-wise PPE detection
Multi-camera monitoring
Real-time alert system (Email/WhatsApp)
Dashboard for analytics
Industrial equipment safety monitoring

💡 Vision
To build a fully automated Industrial Safety Intelligence Platform that helps organizations reduce workplace accidents and achieve zero-incident environments.
